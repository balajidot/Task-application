# Task Planner — EXE Build Instructions

## Step 1: Install Node.js
Download from: https://nodejs.org (LTS version)

## Step 2: Open this folder in Terminal / CMD
Right-click the folder → "Open in Terminal"
OR open CMD and type:
  cd path\to\task-app

## Step 3: Install packages
  npm install

## Step 4: Build EXE
  npm run electron:build

## Step 5: Find your EXE
After build completes, go to:
  release\ folder → Task Planner Setup.exe

Double-click to install and run!

---
Note: First build may take 5-10 minutes (downloads Electron).
